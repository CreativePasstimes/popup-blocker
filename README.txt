# Popup Blocker (Strawberry Jam Plugin)

Blocks annoying login popups (daily gift, diamond spin, news, etc.) in Animal Jam Classic.

## Installation
1. Open Strawberry Jam → Plugins → Add from GitHub.
2. Paste this link: https://github.com/USERNAME/popup-blocker
3. Enable the plugin in the UI.
4. Use the toggle checkbox to enable/disable popup blocking.

## Notes
- Blocks `%xt%dl%`, `%xt%al%`, `%xt%bl%` packets.
- Blocks selected `%xt%gl%` popup/news IDs.
- Toggle lets you turn blocking on/off without uninstalling.
